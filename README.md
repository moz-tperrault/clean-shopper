# 🌿 Clean Shopper

**Clean Shopper** is an AI-powered product research assistant that helps users find clean, non-toxic home and personal care products.

## Overview

Clean Shopper makes it easy to research ingredients, compare products, and make informed choices about what you bring into your home — powered by AI.

## Features (Planned)

- 🔍 AI-powered product search and analysis
- 🧪 Ingredient safety scoring and flagging
- 📋 Product comparison tool
- 🏷️ Category browsing (cleaning, skincare, baby, etc.)
- 💾 Save and organize favorite products
- 📱 Responsive design for mobile and desktop

## Tech Stack

- **Frontend:** React 18, React Router
- **AI:** Anthropic Claude API
- **Styling:** CSS Modules / TailwindCSS (TBD)

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm start

# Run tests
npm test

# Build for production
npm run build
```
